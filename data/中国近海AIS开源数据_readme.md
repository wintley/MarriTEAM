## **背景描述**
中国近海AIS开源数据，近年来中国近海真实的船舶轨迹数据

## **数据说明**
数据由东海船舶轨迹数据（脱敏）编制而成，是真实的海上历史船舶轨迹数据。数据集涵盖多个维度的信息。每个轨迹数据包括渔船ID、经纬度、速度、方向、时间、作业方式（包括拖网、围网和刺网）等。
